to_binary_list(value)

Convert the given hexadecimal value to a list of it's binary representation

Example

iex> Convert.Hexadecimal.to_binary_list("A0")
'10100000'
iex> Convert.Hexadecimal.to_binary_list('A0')
'10100000'
iex> Convert.Hexadecimal.to_binary_list(["A", "0"]) 
'10100000'
