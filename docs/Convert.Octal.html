to_binary_list(value)

Convert the given octal value to a list of it's binary representation

Example

iex> Convert.Octal.to_binary_list(240)

iex> Convert.Octal.to_binary_list("240")

iex> Convert.Octal.to_binary_list('240')

iex> Convert.Octal.to_binary_list([2, 4, 0])
